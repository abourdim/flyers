# 📄 Flyers

**Promotional flyer templates and social media posts for Workshop-DIY apps.**

## Contents

| File | Description |
|------|-------------|
| `flyer-face-quest.html` | Face Quest app flyer |
| `flyer-face-tracking.html` | Face Tracking app flyer |
| `001_bit-bot.html` | Bit-Bot flyer v1 |
| `003_bit-bot.html` | Bit-Bot flyer v3 |
| `template-flyer.html` | Reusable flyer template |
| `*.txt` | Facebook post captions |
| `*.png` | Rendered flyer images |

## Usage

Open any `.html` flyer in a browser, then print or screenshot.

Use the `.txt` files as ready-to-paste Facebook post captions.

## Design

- 1080px width (social media optimized)
- Islamic geometric patterns
- Workshop-DIY branding and QR codes
- Gold/Teal/Blue/Violet color themes
